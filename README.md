# Bose Landing Page

A responsive landing page for **Bose** — built with pure HTML and SCSS, bundled with Parcel.

## 🔗 Demo

> [Live Demo](#)

## 🛠 Tech Stack

- **HTML5** — semantic markup
- **CSS3 / SCSS** — styling with variables, nesting and partials
- **Parcel** — module bundler and dev server

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v14+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/maksother/landing-page.git

# Navigate to the project folder
cd landing-page

# Install dependencies
npm install
```

### Running locally

```bash
npm start
```

The app will open at `http://localhost:1234`

### Build for production

```bash
npm run build
```
